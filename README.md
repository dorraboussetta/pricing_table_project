# Flexbox Pricing Table

A responsive pricing table styled with CSS Flexbox.

## Overview

This project is a Flexbox practice exercise. The HTML structure was provided by the instructor, and my task was to style the pricing table using CSS Flexbox.

The layout includes three pricing plans displayed in a flexible row layout, centered horizontally and vertically on the page. The design also adjusts for smaller screens using a media query.

## Technologies Used

* HTML
* CSS
* Flexbox

## What I Practiced

* Using `display: flex`
* Centering elements with `justify-content` and `align-items`
* Creating spacing with `gap`
* Controlling item sizing with `flex`
* Using `max-width`
* Applying responsive behavior with media queries
